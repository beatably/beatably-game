# Beatably Frontend

React 19 SPA built with Vite. See the root [README](../README.md) and [Architecture](../docs/ARCHITECTURE.md) for full project docs.

## Dev

```bash
npm install
npm run dev    # http://localhost:5173
npm run build  # output: dist/
```

Requires the backend running on `:3001`. See [Deployment](../docs/DEPLOYMENT.md) for setup.
